## 플레이 방법

* 방향키로 0 번 플레이어를 움직입니다.
* q로 게임을 강제종료 합니다.

* 줄다리기 게임
   * 기본 조작키(게임마스터)
      * ‘q’: 줄다리기 종료
      * ‘z’: 왼쪽으로 당김(str 1 감소)
      * ‘/’: 오른쪽으로 당김(str 1 증가)
      * ‘x’: 왼쪽 팀 눕기
      * ‘.’: 오른쪽 팀 눕기
